Test Input Files:

In order to allow for easy adding and/or changing of tests cases, input files were used.

There are 3 text files that contain encoded tests. Each text file corresponds to test_generate_eratothenes_list, test_remove_composites and test_PrimeNumbers

The encoding is as follows:

Input(parameter);Dictionary_Index;Expected_Output;Output_Type

Output_Type can take on values:

0 - List Object
1 - Filled List
2 - String
3 - int

Example:
self.assertEqual(Lab2.generate_eratothenes_list(3)["exit_code"],1)

Encoded as: 

3;exit_code;1;3
