public void TestGetApplication()
This scenario tests if a valid applicant is returned when a user enters a relevant applicant id. This test ensures that users can view applications seemlessly when required.

This scenario tests whether the correct number of files is returned when the user enters a vlaid application id. This test ensures that all corresponding applicant documents can be viewed by the user.

This scenario tests whether a PGO can add an applicaton to the system without any errors. This ensures that an application can be correctly added to the system.

This scenario tests if a valid supervisor is returned when requested by a supervisor id by a PGO. This ensures that supervisors can be correctly allocated and notified when required.
